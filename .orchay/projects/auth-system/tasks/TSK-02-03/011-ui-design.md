## 2. 디자인 시스템

### 2.1 사용 컴포넌트 (Ant Design 6.x)

| 컴포넌트 | 용도 |
|----------|------|
| Card | 폼 컨테이너 |
| Form | 입력 폼 |
| Input.Password | 비밀번호 입력 |
| Input | 이메일 입력 |
| Button | 액션 버튼 |
| Alert | 상태 메시지 |
| Progress | 비밀번호 강도 |
| Result | 완료 상태 |
| Typography | 제목, 설명 |
| Space | 레이아웃 |
| List | 정책 표시 |

### 2.2 색상 체계

| 요소 | 색상 | 용도 |
|------|------|------|
| 주요 버튼 | primary (blue) | 변경/재설정 |
| 성공 메시지 | success (green) | 완료 알림 |
| 에러 메시지 | error (red) | 실패 알림 |
| 정보 메시지 | info (blue) | 안내 메시지 |
| 비밀번호 강도 (약) | red | Progress |
| 비밀번호 강도 (중) | orange | Progress |
| 비밀번호 강도 (강) | green | Progress |

### 2.3 아이콘 사용

| 아이콘 | 용도 |

| CheckCircleOutlined | 정책 충족 |
| CloseCircleOutlined | 정책 미충족 |

## 4. 비밀번호 강도 표시

### 4.1 강도 계산 기준

| 강도 | 색상 | 조건 |
|------|------|------|
| 매우 약함 | red | 1개 이하 정책 충족 |
| 약함 | orange | 2-3개 정책 충족 |
| 보통 | yellow | 4개 정책 충족 |
| 강함 | green | 5개 정책 모두 충족 |

### 4.2 Progress 컴포넌트

```typescript
// 비밀번호 강도 Progress
<Progress
  percent={strength * 20} // 0-100
  strokeColor={strengthColor}
  showInfo={false}
  size="small"
/>
<Typography.Text type={strengthTextType}>
  강도: {strengthLabel}
</Typography.Text>
```

## 6. 인터랙션

### 6.1 로딩 상태

- 버튼: loading prop + 텍스트 변경
- 입력 필드: disabled 상태

### 6.2 실시간 검증

- 비밀번호 입력 시 정책 목록 실시간 업데이트
- 강도 Progress 실시간 변경
- 비밀번호 확인 불일치 시 즉시 에러 표시

### 6.3 에러 처리

| 상황 | 처리 방식 |
|------|----------|
| 현재 비밀번호 틀림 | Alert error + 필드 에러 |
| 정책 미충족 | 정책 목록 빨간색 표시 |
| 비밀번호 불일치 | 필드 하단 에러 메시지 |
| 이전 비밀번호 재사용 | Alert warning |
| 토큰 만료 | Result warning 화면 |
