## 1. 디자인 시스템

### 1.1 사용 컴포넌트 (Ant Design 6.x)

| 컴포넌트 | 용도 |
|----------|------|
| Table | 사용자 목록 |
| Modal | 등록/수정 폼 |
| Form | 입력 폼 |
| Input | 텍스트 입력 |
| Select | 드롭다운 선택 |
| Checkbox | 역할 선택 |
| Tag | 역할 표시 |
| Badge | 상태 표시 |
| Button | 액션 버튼 |
| Space | 레이아웃 |
| Dropdown | 액션 메뉴 |
| Popconfirm | 확인 다이얼로그 |
| message | 토스트 알림 |

### 1.2 색상 체계

| 요소 | 색상 | 용도 |
|------|------|------|
| 활성 상태 | success (green) | Badge |
| 비활성 상태 | default (gray) | Badge |
| 잠금 상태 | error (red) | Badge |
| 주요 버튼 | primary (blue) | 등록, 저장 |
| 위험 버튼 | danger (red) | 삭제, 잠금 |

### 1.3 아이콘 사용

| 아이콘 | 용도 |
|--------|------|
| PlusOutlined | 등록 버튼 |
| EditOutlined | 수정 |
| DeleteOutlined | 삭제 |
| LockOutlined | 잠금 |
| UnlockOutlined | 잠금 해제 |
| KeyOutlined | 비밀번호 초기화 |
| TeamOutlined | 역할 할당 |
| SearchOutlined | 검색 |
| MoreOutlined | 액션 메뉴 |

## 3. 반응형 설계

### 3.1 브레이크포인트

| 화면 크기 | 레이아웃 |
|----------|---------|
| Desktop (≥1024px) | 전체 테이블, 모든 컬럼 표시 |
| Tablet (768-1023px) | 일부 컬럼 숨김 (부서) |
| Mobile (<768px) | 카드형 목록, 액션 버튼 하단 |

## 4. 인터랙션

### 4.1 로딩 상태

- 테이블 로딩: Spin 컴포넌트
- 버튼 로딩: Button loading prop
- 모달 저장: 버튼 로딩 + disabled

### 4.2 에러 처리

- 폼 필드 에러: 필드 하단 빨간색 텍스트
- API 에러: message.error() 토스트

### 4.3 성공 알림

- 등록/수정/삭제: message.success() 토스트
