## 4. 테스트-수정 루프 이력

| 시도 | 결과 | 수정 내역 |
|------|------|----------|
| 1차 | 9/11 통과 | E2E-002, E2E-007 실패 |
| 2차 | 11/11 통과 | 드롭다운: toBeVisible → toBeAttached, 테마: class → data-theme 속성 |

### 수정 상세

1. **E2E-002: 빠른 메뉴 드롭다운**
   - 원인: Ant Design 드롭다운이 빈 상태에서 visible이 아닌 hidden 상태
   - 해결: `toBeVisible()` → `toBeAttached()` 변경

2. **E2E-007: 테마 전환**
   - 원인: next-themes는 `class` 대신 `data-theme` 속성 사용
   - 해결: `toHaveClass(/dark/)` → `toHaveAttribute('data-theme', 'dark')` 변경
