## 2. 화면 전환 흐름

```mermaid
stateDiagram-v2
    [*] --> Loading: 화면 진입
    Loading --> Default: 데이터 로드 완료
    Loading --> Empty: 데이터 없음
    Loading --> Error: 로드 실패

    Default --> SearchExpanded: 더보기 클릭
    SearchExpanded --> Default: 접기 클릭

    Default --> Loading: 검색 실행
    SearchExpanded --> Loading: 검색 실행

    Default --> Selected: 행 체크박스 클릭
    Selected --> Default: 선택 해제
    Selected --> Selected: 추가 선택

    Default --> Loading: 페이지 이동
    Selected --> Loading: 페이지 이동

    Empty --> Loading: 조건 변경 후 검색
    Error --> Loading: 재시도

    Selected --> ConfirmDialog: 삭제 버튼 클릭
    ConfirmDialog --> Loading: 확인
    ConfirmDialog --> Selected: 취소
```

| Default | 행 클릭 | (상세 화면 이동) | onRowClick 콜백 호출 |

| SearchExpanded | 검색 클릭 | Loading | 확장된 조건으로 검색 |

## 3. 화면별 상세

### 3.1 SCR-01: 목록 화면 기본 상태

**화면 목적:**
목록(조회) 화면의 표준 레이아웃. 상단에 검색 조건 Card, 하단에 액션 버튼과 DataTable이 배치된다.

**레이아웃:**

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│ [검색 조건 영역 - Card]                                                          │
│ ┌─────────────────────────────────────────────────────────────────────────────┐ │
│ │  검색조건1         검색조건2         검색조건3           [더보기 v]         │ │
│ │  ┌─────────────┐   ┌─────────────┐   ┌─────────────────────────────────┐   │ │
│ │  │             │   │             │   │ 2026-01-01 ~ 2026-01-21         │   │ │
│ │  └─────────────┘   └─────────────┘   └─────────────────────────────────┘   │ │
│ │                                                                             │ │
│ │                                                    [초기화]  [검색]        │ │
│ └─────────────────────────────────────────────────────────────────────────────┘ │
│                                                                                  │
│ [액션 영역]                                                                      │

│                                                                                  │
│ [그리드 영역 - DataTable]                                                        │
│ ┌─────────────────────────────────────────────────────────────────────────────┐ │
│ │  □   │ 컬럼1 ↕     │ 컬럼2 ↕     │ 컬럼3 ↕     │ 컬럼4 ↕     │ 작업       │ │

│ │  □   │ 데이터      │ 데이터      │ 데이터      │ 데이터      │ [수정]     │ │
│ │  □   │ 데이터      │ 데이터      │ 데이터      │ 데이터      │ [수정]     │ │
│ │  □   │ 데이터      │ 데이터      │ 데이터      │ 데이터      │ [수정]     │ │
│ │  □   │ 데이터      │ 데이터      │ 데이터      │ 데이터      │ [수정]     │ │
│ │  □   │ 데이터      │ 데이터      │ 데이터      │ 데이터      │ [수정]     │ │

│ [페이지네이션]                                                                   │
│ ┌─────────────────────────────────────────────────────────────────────────────┐ │
│ │                     [<] [1] [2] [3] ... [13] [>]    [10건/페이지 v]         │ │
│ └─────────────────────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────────┘
```

**컴포넌트 구성:**

| 영역 | Ant Design 컴포넌트 | Props/설정 |
|------|---------------------|------------|
| 검색 Card | `Card` | bordered, size="small" |
| 검색 폼 | `Form` | layout="inline" 또는 Grid 기반 |
| 검색 필드 | `Input`, `Select`, `DatePicker.RangePicker` | 각 필드별 설정 |
| 더보기 버튼 | `Button` | type="link", icon={<DownOutlined/>} |
| 초기화 버튼 | `Button` | type="default" |
| 검색 버튼 | `Button` | type="primary", icon={<SearchOutlined/>} |
| 신규 버튼 | `Button` | type="primary", icon={<PlusOutlined/>} |
| 삭제 버튼 | `Button` | danger, icon={<DeleteOutlined/>}, disabled={!hasSelection} |
| 총 건수 | `Typography.Text` | - |
| 테이블 | `Table` | columns, dataSource, rowSelection, pagination |

**상태 정의:**

| 상태 | 설명 | 시각적 표현 |
|------|------|------------|
| 기본 | 데이터가 로드된 초기 상태 | 모든 요소 활성화, 체크박스 미선택 |
| 검색 중 | 검색 실행 중 | 검색 버튼에 로딩 스피너, 테이블 로딩 오버레이 |
| 정렬됨 | 컬럼 정렬 적용 | 정렬 아이콘 활성화 (파란색) |

**액션 정의:**

| 액션 | 트리거 | 결과 |
|------|--------|------|
| 검색 | 검색 버튼 클릭 / Enter 키 | 조건에 맞는 데이터 조회 |
| 초기화 | 초기화 버튼 클릭 | 검색 조건 리셋 + 전체 데이터 재조회 |
| 신규 | 신규 버튼 클릭 | onAdd 콜백 호출 (등록 화면 이동) |
| 삭제 | 삭제 버튼 클릭 | 확인 다이얼로그 표시 |
| 행 클릭 | 테이블 행 클릭 | onRowClick 콜백 호출 (상세 화면 이동) |
| 정렬 | 컬럼 헤더 클릭 | 정렬 적용/변경/해제 |
| 페이지 이동 | 페이지 번호 클릭 | 해당 페이지 데이터 로드 |

### 3.2 SCR-02: 검색 조건 확장 상태

**화면 목적:**
검색 조건이 많은 경우 "더보기"를 클릭하여 숨겨진 검색 조건을 펼친 상태.

**레이아웃:**

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│ [검색 조건 영역 - Card]                                                          │
│ ┌─────────────────────────────────────────────────────────────────────────────┐ │
│ │  검색조건1         검색조건2         검색조건3           [접기 ^]           │ │
│ │  ┌─────────────┐   ┌─────────────┐   ┌─────────────────────────────────┐   │ │
│ │  │             │   │             │   │ 2026-01-01 ~ 2026-01-21         │   │ │
│ │  └─────────────┘   └─────────────┘   └─────────────────────────────────┘   │ │
│ │                                                                             │ │
│ │  검색조건4         검색조건5         검색조건6                              │ │
│ │  ┌─────────────┐   ┌─────────────┐   ┌─────────────┐                       │ │
│ │  │             │   │             │   │             │                       │ │
│ │  └─────────────┘   └─────────────┘   └─────────────┘                       │ │
│ │                                                                             │ │
│ │                                                    [초기화]  [검색]        │ │
│ └─────────────────────────────────────────────────────────────────────────────┘ │
│                                                                                  │
│ ... (액션 영역 + 그리드 영역 동일)                                               │
```

**상태별 변화:**

| 상태 | 버튼 텍스트 | 아이콘 | 검색 조건 |
|------|------------|--------|----------|
| 접힘 (기본) | 더보기 | DownOutlined | 기본 3개 표시 |
| 펼침 | 접기 | UpOutlined | 전체 조건 표시 |

**인터랙션:**

- 더보기/접기 클릭 시 애니메이션으로 부드럽게 전환 (Ant Design Collapse 패턴)
- 확장 영역 높이가 동적으로 변경됨

### 3.5 SCR-05: 행 선택 상태

**화면 목적:**
테이블에서 하나 이상의 행이 선택된 상태. 선택된 행은 배경색이 변경되고 삭제 버튼이 활성화된다.

**레이아웃:**

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│ [검색 조건 영역 - Card] (동일)                                                   │
│                                                                                  │
│ [액션 영역]                                                                      │
│ ┌─────────────────────────────────────────────────────────────────────────────┐ │
│ │  [+ 신규]  [삭제] (활성화)                        3건 선택됨  |  총 125건   │ │
│ └─────────────────────────────────────────────────────────────────────────────┘ │
│                                                                                  │
│ [그리드 영역 - DataTable with Selection]                                         │
│ ┌─────────────────────────────────────────────────────────────────────────────┐ │
│ │  [v] │ 컬럼1 ↕     │ 컬럼2 ↕     │ 컬럼3 ↕     │ 컬럼4 ↕     │ 작업       │ │
│ ├──────┼─────────────┼─────────────┼─────────────┼─────────────┼────────────┤ │
│ │ [v]  │ 데이터      │ 데이터      │ 데이터      │ 데이터      │ [수정]     │ │ <- 선택됨 (배경색)
│ │  □   │ 데이터      │ 데이터      │ 데이터      │ 데이터      │ [수정]     │ │
│ │ [v]  │ 데이터      │ 데이터      │ 데이터      │ 데이터      │ [수정]     │ │ <- 선택됨 (배경색)
│ │  □   │ 데이터      │ 데이터      │ 데이터      │ 데이터      │ [수정]     │ │
│ │ [v]  │ 데이터      │ 데이터      │ 데이터      │ 데이터      │ [수정]     │ │ <- 선택됨 (배경색)
│ └──────┴─────────────┴─────────────┴─────────────┴─────────────┴────────────┘ │
│                                                                                  │
│ [페이지네이션] (동일)                                                            │
└─────────────────────────────────────────────────────────────────────────────────┘
```

**상태별 변화:**

| 요소 | 미선택 상태 | 선택 상태 |
|------|------------|----------|
| 삭제 버튼 | disabled (회색) | 활성화 (빨간색) |
| 선택 개수 | 표시 안함 | "N건 선택됨" 표시 |
| 헤더 체크박스 | 미체크 (□) | 일부 선택 (-) 또는 전체 선택 (v) |
| 행 배경색 | 투명/흰색 | primary-1 (연한 파란색 #e6f4ff) |
| 행 체크박스 | 미체크 (□) | 체크됨 (v) |

**인터랙션:**

| 액션 | 결과 |
|------|------|
| 행 체크박스 클릭 | 해당 행 선택/해제 토글 |
| 헤더 체크박스 클릭 | 현재 페이지 전체 선택/해제 |
| 삭제 버튼 클릭 | 확인 다이얼로그 표시 ("N건의 항목을 삭제하시겠습니까?") |

## 5. 반응형 설계

### 5.1 Breakpoint 정의

| Breakpoint | 크기 | 설명 |
|------------|------|------|
| Desktop (lg) | 1200px+ | 모든 요소 기본 배치 |
| Desktop (md) | 992-1199px | 검색 조건 4열 -> 3열 |
| Tablet | 768-991px | 검색 조건 2열, 테이블 가로 스크롤 |
| Mobile | 767px- | 검색 조건 1열, 테이블 가로 스크롤 |

### 5.2 반응형 동작

| 화면 크기 | 검색 조건 | 테이블 | 액션 버튼 |
|----------|----------|--------|----------|
| Desktop | 한 행에 3-4개 | 전체 컬럼 표시 | 좌우 배치 |
| Tablet | 한 행에 2개 | scroll.x 활성화 | 좌우 배치 |
| Mobile | 한 행에 1개 | scroll.x 활성화, 필수 컬럼만 | 세로 스택 |

### 5.3 검색 조건 그리드

```typescript
// Ant Design Grid 설정
<Row gutter={[16, 16]}>
  <Col xs={24} sm={12} md={8} lg={6}>
    {/* 검색 필드 */}
  </Col>
</Row>
```

| 화면 | Col span | 한 행 개수 |
|------|----------|----------|
| xs (<576px) | 24 | 1개 |
| sm (>=576px) | 12 | 2개 |
| md (>=768px) | 8 | 3개 |
| lg (>=992px) | 6 | 4개 |
