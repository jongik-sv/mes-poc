#### BR-001: 단계별 순차 진행 필수

| 구분 | 내용 |

#### BR-002: 다음 이동 전 유효성 검사 필수

| 구분 | 내용 |
|------|------|
| **PRD 원문** | "단계별 유효성 검사" |
| **설계 표현** | 다음 단계로 이동하기 전 현재 단계의 모든 필수 필드와 유효성 규칙을 검사한다 |
| **구현 위치** | Form.validateFields() |

| **관련 테스트** | UT-008, UT-009, E2E-006 |

| **설계 표현** | 단계 이동 시 이전에 입력한 데이터가 유지되어야 한다 |
| **구현 위치** | WizardContext (data state) |
| **검증 방법** | 이전 버튼으로 돌아갔을 때 데이터 유지 확인 |
| **관련 테스트** | UT-010, E2E-007 |
