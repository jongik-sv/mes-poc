### 2.3 테스트 케이스 상세

#### E2E-01: 화면 초기 로드 시 설비 카드가 표시되어야 함
- **전제 조건**: 로그인 완료, 설비 모니터링 페이지 접근
- **테스트 단계**:
  1. 설비 모니터링 컨테이너(`equipment-monitor`) 표시 확인
  2. 설비 그리드 또는 Empty 상태 확인
  3. 그리드 존재 시 최소 1개 이상 카드 표시 확인
- **결과**: ✅ PASS
- **스크린샷**: e2e-equipment-monitor-001.png

- **테스트 단계**:
  1. 전체 설비 수(`summary-total`) 표시 확인
  2. 가동 설비 수(`summary-running`) 표시 확인
  3. 정지 설비 수(`summary-stopped`) 표시 확인
  4. 고장 설비 수(`summary-fault`) 표시 확인
  5. 점검 설비 수(`summary-maintenance`) 표시 확인

  4. 모든 카드의 상태 배지가 '가동'인지 확인
