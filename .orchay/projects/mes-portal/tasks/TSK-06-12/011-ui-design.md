## 2. 화면 전환 흐름

```mermaid
stateDiagram-v2
    [*] --> DimensionInspection: 페이지 진입 (기본)

    DimensionInspection --> AppearanceInspection: 외관 검사 선택
    DimensionInspection --> FunctionInspection: 기능 검사 선택
    AppearanceInspection --> DimensionInspection: 치수 검사 선택
    AppearanceInspection --> FunctionInspection: 기능 검사 선택
    FunctionInspection --> DimensionInspection: 치수 검사 선택
    FunctionInspection --> AppearanceInspection: 외관 검사 선택

    DimensionInspection --> Validating: 저장 클릭
    AppearanceInspection --> Validating: 저장 클릭
    FunctionInspection --> Validating: 저장 클릭

    Validating --> ValidationError: 유효성 검사 실패
    Validating --> Saving: 유효성 검사 통과
    ValidationError --> Validating: 수정 후 재시도

    Saving --> Success: 저장 완료
    Saving --> Error: 저장 실패
    Success --> [*]: 폼 초기화
    Error --> DimensionInspection: 재시도 대기
```
