# TSK-01-01 사용자 매뉴얼 - 역할그룹 정의 화면

## 1. 화면 개요

역할그룹 정의 화면은 RBAC(역할 기반 접근 제어) 시스템의 핵심 관리 화면입니다. 역할그룹을 생성하고, 역할그룹에 역할을 할당하며, 각 역할에 세부 권한을 할당하는 3단계 관리를 하나의 화면에서 수행할 수 있습니다.

화면은 3-column 구조로 구성되어 있으며, 좌측에서 우측으로 계층적으로 탐색합니다.

## 2. 접근 경로

**메뉴 경로:** 시스템 관리 > 역할그룹 정의

**URL:** `/system/role-groups`

## 3. 기능 설명

### 3.1 역할그룹 목록 (좌측 패널, 35%)

#### 역할그룹 조회
1. 페이지 진입 시 전체 역할그룹 목록이 테이블에 표시됩니다.
2. 테이블 컬럼: 코드, 이름, 시스템, 상태, 액션

#### 검색 및 필터
1. 상단 검색창에 역할그룹 이름 또는 코드를 입력하여 검색합니다.
2. "시스템" 드롭다운으로 특정 시스템에 속한 역할그룹만 필터링합니다.
3. "상태" 드롭다운으로 활성/비활성 역할그룹을 필터링합니다.

#### 역할그룹 등록
1. [등록] 버튼을 클릭합니다.
2. 모달에서 역할그룹 코드, 이름, 시스템, 설명, 활성상태를 입력합니다.
3. [저장]을 클릭하여 등록을 완료합니다.

#### 역할그룹 수정/삭제
- 테이블 액션 열의 수정 버튼: 모달에서 정보를 수정합니다 (코드는 수정 불가).
- 테이블 액션 열의 삭제 버튼: 확인 팝업 후 삭제합니다.

### 3.2 역할 관리 (중앙 패널, 33%)

#### 역할 조회
1. 좌측 패널에서 역할그룹 행을 클릭합니다.
2. 중앙 패널 상단에 선택된 역할그룹명이 표시됩니다.
3. "할당된 역할" 섹션: 현재 역할그룹에 할당된 역할 목록
4. "전체 역할" 섹션: 시스템의 모든 역할 목록 (체크박스 포함)

#### 역할 할당
1. "전체 역할" 테이블에서 할당할 역할의 체크박스를 선택/해제합니다.
2. 이미 할당된 역할은 체크 상태로 표시됩니다.
3. [할당 저장] 버튼을 클릭하여 변경사항을 저장합니다.
4. 저장 성공 시 "할당된 역할" 테이블이 갱신됩니다.

#### 역할 등록/수정/삭제
- [등록] 버튼: 새 역할 생성 모달 (코드, 이름, 설명, 활성상태)
- 할당된 역할 테이블의 수정/삭제 액션 사용

### 3.3 권한 관리 (우측 패널, 32%)

#### 권한 조회
1. 중앙 패널의 "할당된 역할" 테이블에서 역할 행을 클릭합니다.
2. 우측 패널 상단에 선택된 역할명이 표시됩니다.
3. "할당된 권한" 섹션: 현재 역할에 할당된 권한 목록
4. "전체 권한" 섹션: 시스템의 모든 권한 목록 (체크박스 포함)

#### 권한 할당
1. "전체 권한" 테이블에서 할당할 권한의 체크박스를 선택/해제합니다.
2. [할당 저장] 버튼으로 변경사항을 저장합니다.

#### 권한 등록/수정/삭제
- [등록] 버튼: 새 권한 생성 모달 (코드, 이름, 유형[메뉴/API/데이터], 설명, 활성상태)
- 할당된 권한 테이블의 수정/삭제 액션 사용

## 4. 주의사항

- 역할그룹을 선택하지 않으면 중앙 패널은 "역할그룹을 선택하면 역할 목록이 표시됩니다." 안내를 표시합니다.
- 역할을 선택하지 않으면 우측 패널은 "역할을 선택하면 권한 목록이 표시됩니다." 안내를 표시합니다.
- 다른 역할그룹을 선택하면 우측 패널(권한)은 초기화됩니다. 역할을 다시 선택해야 합니다.
- 역할그룹/역할/권한 삭제 시 연관된 할당 관계도 함께 삭제됩니다.
- 할당 저장은 체크박스 변경 후 반드시 [할당 저장] 버튼을 클릭해야 반영됩니다.
- 코드 필드는 등록 시에만 입력 가능하며, 수정 시에는 변경할 수 없습니다.
